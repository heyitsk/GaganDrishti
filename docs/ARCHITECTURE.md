# GaganDrishti – Architecture

GaganDrishti follows a modular architecture to detect and protect
against cloud misconfigurations.

## Overview
GaganDrishti follows a simple modular architecture focused on clarity,
security, and ease of demonstration.

## Components
1. Cloud Scanner Engine (Backend)
2. Detection & Risk Analysis Module
3. API Layer
4. Frontend Dashboard

## Flow
User → Cloud Scanner → Detection Rules → Risk Analysis → Dashboard

## Design Principles
- Read-only cloud access
- No credential storage
- Preventive security focus
