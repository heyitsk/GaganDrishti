# Threat Model – GaganDrishti

## Threat Actors
- External attackers
- Malicious insiders
- Automated bots

## Attack Vectors
- Exploiting public storage
- Brute-forcing exposed ports
- Privilege escalation via IAM

## Security Objective
Prevent attacks by detecting insecure configurations before exploitation.

## Assumption
GaganDrishti operates with read-only access and does not modify cloud
resources.
